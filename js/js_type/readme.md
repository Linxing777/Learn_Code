# js 数据类型

- js有多少种数据类型？
   **内存分配机制**
   **拷贝 引用**
  
  - 简单数据类型
     string number boolean null undefined symbol bigint
    // 依据变量内存的分配机制 栈内存 复杂数据类型的栈里存的是指向堆内存的地址
   
  - 简单类型数据是拷贝，复杂的是引用


  - 复杂数据类型
     object

  - null 
    null 表示一个空值或者不存在的对象 是一个可以赋变量的特殊值
    显示回收内存

    - undefined 未定义 （类型）

    - symbol 独一无二的值 es6新增的数据类型
      - 以函数的形式创建 简单值
      - 打个便签
      - 标签一样，值绝对不一样
    
    - bigint 大整数  任意精度整数

    - 简单数据类型 Primitive
      一眼看出值

      函数、数组等有很多特性无法表达
